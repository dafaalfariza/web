ğŸ”¹ HTML (HyperText Markup Language)



HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar halaman web.

ğŸ‘‰ Bisa dibilang kerangka atau tulang dari website.

  • Contoh elemen: heading (<h1>), paragraf (<p>), gambar (<img>), link (<a>).

  • Tanpa HTML, halaman web tidak punya isi atau struktur.

Contoh:


<!DOCTYPE html>
<body>
<h1>Hello World</h1>
<p>This is my first paragraph</p>
</body>
</html>

ğŸ“Œ Struktur Dasar


<!DOCTYPE html> → deklarasi jenis dokumen HTML.

<html> → elemen root, membungkus semua isi halaman.

<head< → berisi metadata (judul, link css, script, dll).

<titke> → judul halaman (muncul di tab browser).

<body> → isi utama halaman yang tampil ke user.



ğŸ“Œ Teks & Heading


<h1> - <h6> → heading (judul) dari besar → kecil.

<p> → paragraf.

<span> → teks inline (biasa digunakan untuk styling).

<br> → break baris baru.

<hr> → garis horizontal pemisah.



ğŸ“Œ Format Teks


<b> → teks tebal (bold).

<i> → teks miring (italic).

<u> → teks bergaris bawah (underline).

<strong> → teks penting (biasanya tebal, punya arti semantik).

<em> → teks ditekankan (biasanya miring, ounya arti semantik).

<mark> → teks disorot (highlight).

<small> → teks kecil.

<del> → teks coret.

<ins> → teks yang ditambahkan (garis bawah).

<sub> → teks subscript (misal: H2O).

<sup> → teks superscript (misal: x2).



ğŸ“Œ List (Daftar)


<ul> → unordered list (bullet).

<ol> → ordered list (angka/huruf).

<li> → item di dalam list.

<dl> → description list.

<dt> → istilah dalam description list.

<dd> → deskripsi dari istilah.



ğŸ“Œ Link & Media


<a> → hyperlink (link ke html lain).

<img> → menampilkan gambar.

<video> → menampilkan video.

<audio> → menampilkan audio.

<>source → sumber file media (untuk <video> / <audio>).

<>iframe → menyematkan halaman lain (misalnya YouTube).



ğŸ“Œ Table


<table> → membuat tabel.

<tr> → table row (baris).

<td> → table data (kolom biasa).

<th> → table header (kolom judul, biasanya tebal).

<thead> → kepala tabel.

<tbody> → isi tabel.

<tfoot> → kaki tabel.

<caption> → judul tabel.



ğŸ“Œ Formulir (Input Data)


<form> → wadah form.

<input> → field input (text, number, chehckbox, radio, dll).

<textarea> → input teks panjang.

<button> → tombol.

<select> → dropdown.

<option> → opsi dalam dropdown.

<label> → label untuk input.

<fieldset> → kelompokkan elemen dalam form.

<legend> → judul kelompok dalam fieldset.



ğŸ“Œ Script & Style


<style> → menulis CSS langsung di HTML.

<script> → menulis JavaScript.

<link> → menghubungkan file CSS.

<meta> → metadata (charset, viewport, SEO, dll).



ğŸ‘‰ Jadi intinya:

  • Struktur → <html>, <head>, <body>, <main>, dll.

  • Konten → <p>, <h1>, <img>, <a>.

  • Layout → <div>, <section>, <article>.

  • Interaktif → <form>, <button>, <input>.



ğŸ”¹ CSS (Cascading Style Sheets)

CSS adalah bahasa untuk mengatur tampilan (desain) HTML.

ğŸ‘‰ Digunakan untuk mengatur warna, ukuran, font, tata letak, dll.

  • HTML bikin kerangka → CSS bikin tampilannya lebih menarik.

Contoh:


p {
   color: white;
   background-color: gray;
   padding: 10px;
}

ğŸ“Œ Selector Dasar


Selector adalah "target" elemen HTML yang mau diatur.

p { } → untuk semua paragraf.

.class { } → untuk elemen dengan class tertentu.

;#id ( ) → untuk elemen dengan id tertentu.



ğŸ“Œ Property Dasar


Beberapa properti CSS yang sering dipakai:

<color> → warna teks.

<background-color> → warna latar.

<font-size> → ukuran huruf.

<font-family> → jenis huruf.

<margin> → jarak luar elemen.

<padding> →jarak dalam elemen.

<border> → garis tepi.
